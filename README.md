# Drive Project

**Apache + PHP + MySQL**

### 계획
  - 가상 드라이브 만들기 ex) 구글 드라이브
  - Apache + PHP + MySQL 연동
  - 기초적인 홈페이지 만들기
    - HTML + CSS + JavaScript 사용
    - 로그인, 회원가입, 아이디/비밀번호 찾기 구현  
  - 회원가입한 사용자 마다 서버에 폴더를 생성
    - 서버 상에 사용자의 아이디로 루트 폴더 생성
    - 루트 폴더 내에 사용자가 폴더 생성 및 파일 업로드/다운로드
